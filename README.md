### 一、创建 vue3.0+vite 项目

```
$ yarn create vite-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev

```

### 二、启动项目

```
npm install
npm run dev
npm run build
```

### 三、简介

```
vite —— 一个由 vue 作者尤雨溪开发的 web 开发工具，它具有以下特点：
	快速的冷启动
	即时的模块热更新
	真正的按需编译
```

添加 vite.config.js 配置文件

### 四、安装路由
```
npm install vue-router@4.0.0-beta.6 -S
```
